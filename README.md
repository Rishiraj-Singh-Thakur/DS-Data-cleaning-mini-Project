🧹 Data Cleaning Mini Project
📌 Overview

This project is part of my Data Science journey where I worked on cleaning and processing raw JSON data. The goal was to transform unclear and messy data into structured and usable format.

🚀 Features

Clean and remove duplicate/unclear entries
Structure data properly
Save cleaned data into a new JSON file
Simple and beginner-friendly implementation

🛠️ Technologies Used
Python 🐍

Jupyter Notebook 📓

JSON

⚙️ How It Works

Load raw data from unclearData.json
Process and clean the data
Remove duplicates and unwanted values
Store cleaned data in clear_data.json

📊 Use Cases

Data preprocessing
Beginner Data Science projects
Learning JSON handling in Python
